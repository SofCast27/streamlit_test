# Streamlit Data App

## Descripción
Aplicación interactiva desarrollada con Python y Streamlit que simula el lanzamiento de una moneda
para analizar probabilidades de ocurrencia.

La aplicación permite ejecutar múltiples intentos, registrar los resultados de cada lanzamiento
y visualizar de forma dinámica cómo evolucionan las probabilidades a medida que aumenta el número
de simulaciones. A través de gráficos generados en tiempo real, se observa la convergencia de los
resultados hacia la probabilidad teórica, facilitando la comprensión de conceptos básicos de
estadística y análisis probabilístico.

## Ejecución local
1. Clonar el repositorio
2. Instalar dependencias:
   pip install -r requirements.txt
3. Ejecutar la aplicación:
   streamlit run app.py
